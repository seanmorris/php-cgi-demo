Omni Explorer Help

What's this?

Omni Explorer allows you to browse & modify the files available to php-wasm.

Note:If no directories are displayed, open up the php cgi service worker app and ensure the service is running, then restart Omni Explorer.

Directory Types

  • This is a normal directory within the filesystem.
  • This representes a mounted directory.

Top level directories like /tmp or /home are regenerated each time php-wasm is re-instantiated. Since php-cgi-wasm runs in the service worker, this won't be on ever page refresh but its important to remember that these folders cannot store anything persistently.

Mounted directories that are backed by persistent IDBFS volumes are marked with a icon. These directories will contain PHP source code and configuration files, so they can be stored and modified without fear of losing them.

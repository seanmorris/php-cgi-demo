php-cgi-wasm × Nynex95℠

What's this?

php-cgi-wasm × Nynex95℠ is a demo of the capabilities of php-cgi-wasm using the Nynex95℠ prototyping GUI.

php-cgi-wasm runs the php-cgi binary inside a ServiceWorker, under web assembly. This allows normal PHP code to run in response to HTTP requests, which are intercepted and handled by the ServiceWorker.

Nynex95℠ is simply a prototyping GUI I've put together.

Applications:

Package Installer

Install a PHP package such as Drupal, Laravel or Laminas. You can also backup, restore, or clear the entire filesystem.

CGI Service Worker

This application allows you to start/stop the CGI Service Worker, as well as change its settings.

Omni Explorer

Omni Explorer allows you to browse & modify the files available to php-wasm.

Sites

Sites allows you to list & access the packages installed under php-cgi-wasm at their respective URLs.
